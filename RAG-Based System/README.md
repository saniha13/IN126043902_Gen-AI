# RAG Customer Support Assistant

## Project Description
This project is a Retrieval-Augmented Generation based Customer Support Assistant built using LangGraph and HITL.

It answers customer queries using a PDF knowledge base.

## Features
- PDF document processing
- Retrieval-based answers
- Customer support chatbot
- Human-in-the-Loop escalation
- Streamlit web interface

## Technologies Used
- Python
- Streamlit
- LangChain
- LangGraph
- FAISS
- PyPDF

## Project Files
app.py
requirements.txt
knowledge_base.pdf

## How to Run

pip install -r requirements.txt

python -m streamlit run app.py

## Sample Questions
- What is return policy?
- How can I track my order?
- How to contact support?
